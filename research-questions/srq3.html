SRQ3 — What is SaaS, and how can CRM platforms be offered as a SaaS?

Software as a Service (SaaS) is a form of cloud computing where user needs are hosted within the cloud as ‘microservices’, making it accessible at any given time for the users via the internet. Its popularity revolves around the digitalization of businesses — instead of self-maintenance and installation from the user part, all necessary apps are hosted by the cloud provider. Data security will not be compromised, as there is one unique entry point necessary from within the cloud.


CRM platforms can be offered as a SaaS by hosting the software via the internet, enabling users to utilize the microservices offered within the software. Its multi-tenant architecture allows the software to work for multiple tenants, allowing efficient resource utilization and stability. Due to its availability within the cloud, it is required to be a web-based application.



The Approach: How?

This answer was gathered mainly from literature study of CRM and SaaS, and the competitive analysis that follows — identifying SaaS components or aspects that concurr within a CRM system. Card sorting session with Obero’s technical advisor and data analyst helps contributing the real life use cases of these microservices within Obero’s own management platform, and a realization that the platform itself does not resemble a full CRM system, and that CRM systems offer various different solutions that usually is tailor-made by the request of each subscribers.
